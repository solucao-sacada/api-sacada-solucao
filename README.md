## Installation

```bash
$ npm install
```

## Docker

```bash
$ docker compose up -d
```

## Migrations

```bash
# up migrations in database
$ npm run migrate
```
## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Test

```bash
# e2e tests
$ npm run test:e2e

# test units
$ npm run test:unit
```
